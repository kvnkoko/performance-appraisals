# Performance Appraisal System

A comprehensive, production-ready performance appraisal web application built with React, TypeScript, and Tailwind CSS.

## Features

- **4 Appraisal Types**: Executives→Leaders, Leaders→Members, Members→Leaders, Leaders→Leaders
- **Template Management**: Create customizable appraisal templates with weighted questions
- **Link Distribution**: Generate secure, unique links for employee appraisals
- **Analytics & Scoring**: Real-time calculations and 6-month review dashboards
- **AI Summaries**: Automated performance summaries with actionable insights
- **Employee Recognition**: Leaderboards and Employee of the Month
- **Dark/Light Mode**: Beautiful, modern UI with smooth transitions
- **Fully Offline**: Works entirely client-side with IndexedDB storage

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- React Hook Form + Zod
- Recharts
- IndexedDB (idb)
